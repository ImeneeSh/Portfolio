# Personal Portfolio – Nour El Imene Sahi

## Overview

This project is a personal portfolio website designed to showcase my profile, skills, projects, and experience as a Computer Science engineering student specializing in Software Engineering.

The goal of this portfolio is to provide a clean, modern, and interactive interface that reflects both my technical abilities and my design sensibility.

---

## Features

- Responsive design for desktop and mobile devices
- Interactive animated background with dynamic effects
- Dark / Light mode toggle
- Bilingual support (French / English)
- Smooth scrolling navigation
- Animated UI elements (hover effects, transitions)
- Project filtering system (Projects / Certifications)
- Downloadable CV integration

## Key Functionalities

### 1. Theme Management (Dark / Light Mode)
Users can toggle between dark and light themes dynamically using JavaScript.

### 2. Language Switching
The interface supports both French and English using `data-*` attributes and dynamic content replacement.

### 3. Interactive Background
- Animated gradient background
- Mouse-based parallax effect
- Dynamic star generation

### 4. Responsive Navigation
- Desktop navigation bar
- Mobile burger menu with smooth transitions

### 5. Scroll Behavior
- Smooth scrolling between sections
- Automatic scroll reset on page reload
